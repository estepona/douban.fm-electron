## 功能
- 登入 / 登出，cookie 存储
- 频道:
  - 豆瓣精选
  - 豆瓣推荐
  - 红心歌曲
  - 私人频道
- 应用置顶
- ...

## 安装

安装包可以在 [Release 页面](https://github.com/estepona/douban.fm-electron/releases) 找到，目前支持 Mac 及 Windows 平台。

## 快捷键
- 播放 / 暂停: <kbd>space</kbd>
- 下一首: <kbd>right</kbd>
- 加入红心歌曲: <kbd>up</kbd>
- 删除红心歌曲: <kbd>down</kbd>
- 刷新应用: <kbd>Ctrl</kbd> + <kbd>R</kbd> (Win / Linux), <kbd>Cmd</kbd> + <kbd>R</kbd> (Mac)
- 置顶应用: <kbd>Ctrl</kbd> + <kbd>E</kbd> (Win / Linux), <kbd>Cmd</kbd> + <kbd>E</kbd> (Mac)
- 重启应用: <kbd>Ctrl</kbd> + <kbd>L</kbd> (Win / Linux), <kbd>Cmd</kbd> + <kbd>L</kbd> (Mac)
- 退出应用: <kbd>Ctrl</kbd> + <kbd>Q</kbd> (Win / Linux), <kbd>Cmd</kbd> + <kbd>Q</kbd> (Mac)

## 已知问题

- 目前仅支持账号与密码登录，不支持手机或第三方登录；
- 如要查看【从单曲出发】，需登录后重启应用。
